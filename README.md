# News-App

News search application provides trending news, all news and news according to user search. The News Search API lets you send a search query to Bing and get back a list of news that are relevant to the search query. Users can get all kinds of news according to their need easily on a single platform, whenever they want and wherever they want. This app makes it easier for them to access news at their fingertips.
